== Description ==

Widget that allow the user to create a custom loop.

== Installation ==

1. Upload `widget-custom-loop` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag'n Drop the `Custom Loop` widget to the sidebar
